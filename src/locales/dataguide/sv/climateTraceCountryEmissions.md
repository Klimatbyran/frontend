# Totala utsläpp

Detta är det senaste tillgängliga året för CO₂e-utsläpp per capita för landet, från Climate TRACE.

Climate TRACE kombinerar satellitdata, markmätningar och rapporterade inventeringar för att uppskatta utsläpp. Värdena kan skilja sig från officiellt rapporterade nationella totaler. För Sveriges officiella territoriella sektorsuppdelning, se sektordiagrammet längre ner.
