# Varför relatera till skogsbränder?

För att sätta utsläpps­förändringen i ett konkret sammanhang räknar vi ut hur stor markyta som skulle behöva brinna för att generera motsvarande mängd utsläpp. Vi utgår från följande antaganden:

- I genomsnitt innehåller varje hektar skogsmark omkring 50 ton kol (C). (1)
- Kol omvandlas till koldioxid (CO₂) genom faktorn 44/12 (molekylvikter), vilket innebär att en hektar motsvarar ungefär 50 × (44/12) ≈ 183 ton CO₂.

Utifrån utsläpps­förändringen räknar vi ut hur många hektar som krävs för att motsvara den mängden CO₂. Därefter jämförs denna area med välkända referenser som fotbollsplaner, tennisbanor eller stads­ytor, för att ge användaren en intuitiv bild.Tack vare denna metod kan vi säga något i stil med: “Det här utsläppet motsvarar att X fotbollsplaner brinner.” På så sätt blir annars abstrakta siffror mer konkreta och relaterbara.
