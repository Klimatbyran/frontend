# Varför använda TCRE-ramverket?

**Transient Climate Response to Cumulative Emissions (TCRE)** är ett vetenskapligt beprövat ramverk som visar ett direkt, linjärt samband mellan den totala mängden CO₂ som släpps ut och den resulterande globala temperaturökningen. Det gör att vi kan beräkna exakt hur mycket ett företags ackumulerade utsläpp bidrar till global uppvärmning i milligrader Celsius (m°C).

## Hur det visar ett företags bidrag

Med TCRE-formeln kan vi ta ett företags totala historiska utsläpp (t.ex. 1,61 GtCO₂) och omvandla dem direkt till ett temperaturvärde (t.ex. 0,72 m°C).

- **Det skapar ansvarighet:** Samtalet flyttas från "ton gas" (som är svårt att visualisera) till "grader uppvärmning".
- **Det är en permanent post:** Eftersom CO₂-utsläpp stannar kvar i atmosfären i århundraden representerar TCRE ett företags "permanent avtryck" på jordens termostat.

## Ackumulerat vs. årligt

Denna graf visar **ackumulerad** data. Till skillnad från årliga mål minskar ackumulerade utsläpp aldrig om inte kol fysiskt avlägsnas från atmosfären.

## Milligrader (m°C)

Även om 0,7 m°C kan låta litet för ett företag (som ABB) finns det tusentals stora företag. Tillsammans skapar dessa "milligrader" den totala globala uppvärmningen på 1,5°C till 2,0°C.
